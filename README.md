# Online-Reservation-System
The Online Reservation System is a Java-based application developed to simplify the process of booking and managing travel reservations through an easy-to-use digital platform. The project includes essential modules such as secure user login authentication, ticket reservation, and reservation cancellation. Users can enter travel details like train number, journey date, source, destination, and class type, while the system automatically retrieves related train information to improve accuracy and reduce manual effort. The application also supports ticket cancellation using a PNR number, where users can view reservation details before confirming cancellation. Built using core Java concepts and database connectivity, the project demonstrates practical implementation of form handling, user authentication, data management, and reservation processing in a real-world transportation booking scenario. The system is designed to reduce manual workload, improve booking efficiency, and provide a user-friendly experience for passengers and administrators.
